## 缺失原因说明

| 算子 | 缺失项 | 原因 |
|------|--------|------|
| div | PyPTO 无数据 | 广播 (broadcast) 触发 PyPTO 后端 CompileFunction 失败，无法生成有效 kernel |
| equal | PyPTO 无数据 | RC-2 虽已解封 (修复 DT_BOOL 输出 + ta≤64 约束)，但未完成 msprof 性能采集 |
| where | PyPTO 无数据 | RC-2 解封 (uint8→bool 修复)，但未完成 msprof 性能采集 |
| transpose | PyPTO 无数据 | tile_shape 对 >1K 元素失效；RC-2 虽修复 (64×256) 但未完成 msprof 性能采集 |
| reduce_sum | PyPTO 无数据 | 使用 wrapper FP32 cast 方案，未进行独立 msprof 性能采集 |
| matmul | PyPTO 无数据 | 自动 tiling 完全损坏 (FC4000)；4D JIT 编译大 shape 超时 |
