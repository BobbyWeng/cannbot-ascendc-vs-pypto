# RC-3 Dashboard Report

## Dashboard v2 — 10 New Features

### 1. Profiler Timeline
Per-operator, per-route kernel timeline with name, type, and duration.

### 2. Batch Scaling
Latency vs batch size visualization for all 3 routes.

### 3. Kernel Timeline
Kernel type breakdown (AIVEC, AICORE, AICPU, MIX_AIC) per logical call.

### 4. Skill Trace
SKILL_TRACE status per operator displayed in main table.

### 5. Backend Limitations
Severity-colored table of known limitations per operator.

### 6. Source Hash
SHA256 of source files displayed on hover.

### 7. Binary Hash
SHA256 of build artifacts in operator detail tab.

### 8. Current Release
Version, commit, date displayed in header.

### 9. Release History
Parsed from RELEASE_CHANGELOG.md, 3 versions displayed.

### 10. Operator Detail Page
6-tab layout per operator: Overview, Profiler, Scaling, Kernels, Limitations, Sources.

## Files
- `dashboard/dashboard.json` (v1.3-rc3)
- `dashboard/index.html` (updated)
- `dashboard/dashboard.py` (release mode: `--release reports/release/current_release.json`)

## Generation
Dashboard is generated by:
```bash
python3 dashboard/dashboard.py --release reports/release/current_release.json
```

Or as part of the release pipeline:
```bash
python3 scripts/release/release.py --step=5
```
